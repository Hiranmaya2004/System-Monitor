Installation
------------------------------------------------------------
Prerequisites:
- Linux environment
- C++17 compiler (g++)
- ncurses library

Install dependencies (Ubuntu/Debian):
sudo apt update
sudo apt install g++ libncurses-dev

------------------------------------------------------------
Build and Run
------------------------------------------------------------
Build (Day 5 version):
g++ -std=c++17 sysmon.cpp -o sysmon -lncurses

Run:
./sysmon 2
(refreshes every 2 seconds; press 'q' to quit)
